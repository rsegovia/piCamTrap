# piCamTrap
An open-source wild life camera trap powered by Raspberry Pi, YOLO and Berrynet.

## Requirement
- Hardware
> Raspberry Pi 3 Model B

> picamera module v2 

> Usb or ip camera(not fully tested)

> Optional: HDMI screen, mouse, keyboard

- Software

> Tensorflow

> Darknet

> Berrynet

> Python 3.4+

- Optional: 
> Goolge drive cloud API

> Twitter API

> MQTT server/broker

## Installment

    git clone 
    run /.build
